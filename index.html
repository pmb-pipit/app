Konfigurasi aplikasi SiPMB

Koneksi Google Spreadsheet
Dapatkan URL ini dari Google Apps Script → Deploy → New Deployment → Web App.
Pilih Execute as: Me dan Who has access: Anyone.
Inisialisasi Spreadsheet

Jalankan fungsi setupSpreadsheet() sekali dari Apps Script Editor untuk membuat semua sheet dan akun default.

1. Buka Google Apps Script
2. Paste kode Code.gs
3. Jalankan fungsi: setupSpreadsheet()
4. Deploy sebagai Web App
5. Salin URL dan paste di atas
